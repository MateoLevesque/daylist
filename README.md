# Daylist
Daylist is a simple and easy to use daily task manager TUI with vim motions.
The name is from the combinaison of *daily* and *list*.

## Main features
- Simple month view where you can move trough the days using `hjkl`.
- Move trough months with `JK`
- Add task with `a`
- Remove task with `r`
- Edit task with `e`

## Futur features
- mark tasks as done
- add weekly events
- add monthly events
- add yearly events

## Potential features
- daily tasks (useful for recurring group of tasks)
