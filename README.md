# Daylist
Daylist is a simple and easy to use daily task manager TUI with vim motions.
The name is from the combinaison of *daily* and *list*.

> [!TIP]
> I'm actively looking for feedback on this project. If you have any features you'd like to be added or any tips to improve the code, the UX or anything, feel free to share it with me.


## Main features
- Simple month view where you can move trough the days using `hjkl`.
- Move trough months with `JK`
- Add task with `a`
- Remove task with `r`
- Edit task with `e`

## Futur features
- mark tasks as done
- add daily events
- add weekly events
- add monthly events
- add yearly events

## Potential features I'd like to be included
